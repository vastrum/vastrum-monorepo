# Tailscale

Essentially use Vastrum to connect P2P applications and bootstrap P2P connections



One example for this could be a Tailscale alternative, have a shared session key > write their local endpoints to that shared session key, 
then have some kind of coordinated UDP hole punching to connect.


Could allow for Tailscale like ux when setting up private networks, ie only need session keys and they will connect to each other.


Another example could be for bootstrapping P2P networks, this essentially moves bootstrapping problem from its own application to Vastrum.


This might not be feasible but interesting domain.
