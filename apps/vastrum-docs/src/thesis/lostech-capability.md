# Lostech Capability


Operating the network should in general be very simple. There should not be a large operational maintenance load to keep the network running.

Node operation UX must be very easy
-   Very minimal config (maybe even dont expose any config)
    -   Even just exposing config to change DB storage location might be too much config
-   Quick sync
-   No footguns
-   Built in sanity production stuff such as auto restart
-   Single command to install.
